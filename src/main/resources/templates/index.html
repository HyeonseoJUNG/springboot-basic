1. 고객 관리

(1) 고객 생성하기
    고객 생성하기 버튼 클릭
      > 이름 입력
      > 이메일 주소 입력
      > submit 버튼 클릭

(2) 고객 상세페이지 확인하기
    표에서 확인하고 싶은 고객의 행을 클릭
      > 고객 정보와 바우처 지갑(고객이 보유한 바우처) 확인 가능

(3) 고객 삭제하기
    표에서 삭제하고 싶은 고객의 행을 클릭
      > 고객 상세페이지로 이동
      > 고객 삭제하기 버튼 클릭

(4) 바우처 지갑 삭제하기
    표에서 삭제하고 싶은 고객의 행을 클릭
      > 고객 상세페이지로 이동
      > 바우처 전체 삭제하기 버튼 클릭


2. 바우처 관리

(1) 바우처 생성하기
    바우처 생성하기 버튼 클릭
      > 바우처 타입 선택
      > 할인액/할인율 입력
      > submit 버튼 클릭

(2) 바우처 발급하기
    바우처 발급하기 버튼 클릭
      > 발급할 바우처 선택
      > 발급 대상 입력
      > submit 버튼 클릭

(3) 바우처 상세페이지 확인하기
    표에서 확인하고 싶은 바우처의 행을 클릭
      > 바우처 정보와 해당 바우처를 발급받은 고객 정보 확인 가능

(4) 바우처 삭제하기
    표에서 삭제하고 싶은 바우처의 행을 클릭
      > 바우처 상세페이지로 이동
      > 바우처 삭제하기 버튼 클릭
